
*******************************************************************************
		How to update/sync entities in our project with ACMEPay database
*******************************************************************************

1.	In package manager console, select API.Repository as "Default project"

2.	Run the following scaffold command:
/*
Scaffold-DbContext "Server=.;Database=ACMEPAY;Trusted_Connection=False;user ID=sa; password=123;" Microsoft.EntityFrameworkCore.SqlServer -OutputDir ORM\Entities -ContextDir ORM\Context -Context AppDbContext -force
*/

3.	Open ACMEDbContext and remove the method;
		OnConfiguring(DbContextOptionsBuilder optionsBuilder)

4. Replace the namespaces generated by scaffolding command as;
	API.Repository.ORM.Entities	=>		API.Repository
	API.Repository.ORM.Context	=>		API.Repository
